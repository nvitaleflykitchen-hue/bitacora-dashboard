# Plantilla de bienvenida para nuevos usuarios

Plantilla reutilizable para enviar después de crear una cuenta en Fly Gestión. Antes de enviarla, reemplazar todos los campos entre corchetes y eliminar cualquier línea que no corresponda.

## Datos que se deben completar

- `[NOMBRE]`: nombre de la persona.
- `[ROL]`: función asignada en la plataforma, expresada en lenguaje natural.
- `[SEDE]`: sede, escala o unidad productiva correspondiente.
- `[EMAIL]`: correo utilizado como usuario.
- `[CONTRASEÑA_TEMPORAL]`: contraseña temporal informada por el sistema.
- `[ENLACE_APP]`: enlace vigente de acceso a la aplicación.

## Mensaje listo para WhatsApp o correo

Hola [NOMBRE], ¡bienvenida/o a Fly Gestión! 👋

Te damos acceso a nuestra plataforma de gestión como [ROL] de [SEDE].

La aplicación fue creada para registrar novedades, tareas, controles, necesidades de mantenimiento y demás situaciones relevantes de la operación diaria. Esperamos que los registros sean claros, oportunos y lo más completos posible, ya que esta información nos permite hacer seguimiento, anticiparnos a los problemas y tomar mejores decisiones.

Tu mirada profesional y tu experiencia son muy importantes. Te pedimos que utilices la plataforma no solo para informar inconvenientes, sino también para dejar observaciones, detectar oportunidades de mejora y aportar todo aquello que consideres útil para optimizar el servicio y la organización del equipo.

La plataforma se encuentra en mejora permanente. Por eso, cualquier sugerencia sobre su funcionamiento, información faltante o herramienta que pueda facilitar el trabajo será bienvenida. Queremos que nos lo expreses con total confianza: estamos para escuchar, acompañar y seguir mejorando juntos.

Datos de acceso:

- Aplicación: [ENLACE_APP]
- Usuario: [EMAIL]
- Contraseña temporal: `[CONTRASEÑA_TEMPORAL]`

Al ingresar por primera vez, el sistema te pedirá elegir una nueva contraseña.

¡Bienvenida/o al equipo, [NOMBRE]! Esperamos que esta herramienta sea de gran utilidad para tu gestión diaria.

## Lista de control antes de enviar

- Confirmar que el nombre esté bien escrito.
- Confirmar el rol y la sede asignados en la aplicación.
- Confirmar que el correo coincida con el usuario creado.
- Verificar el enlace vigente de acceso.
- No reutilizar ni conservar una contraseña personal del usuario.
- Enviar el mensaje únicamente a la persona incorporada o por un canal autorizado.

## Ejemplo de personalización

Para una encargada llamada Jaquelina en Aeropuerto Rosario:

- `[NOMBRE]` → `Jaquelina`
- `[ROL]` → `encargada`
- `[SEDE]` → `Aeropuerto Rosario`
- `[EMAIL]` → correo con el que se creó la cuenta
- `[CONTRASEÑA_TEMPORAL]` → contraseña indicada por el alta
- `[ENLACE_APP]` → `https://bitacora-dashboard.vercel.app`
